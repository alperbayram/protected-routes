## Installation
Clone the repo
```bash 
  git clone https://github.com/alperbayram/protected-routes.git
```

Open folder
```bash 
  cd protected-routes
```

Install NPM packages
```bash 
  npm install
```

## Usage

```bash 
  npm start
```
```bash 
  http://localhost:3000
```
